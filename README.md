# Student Time Master ⏰🎓

A web app to help students manage schedules, track tasks, and get AI-powered study assistance.

![Screenshot](https://via.placeholder.com/800x400.png?text=Student+Time+Master+Screenshot) <!-- Replace with your own screenshot -->

## Features ✨
- **Smart Calendar**: Schedule classes, exams, and events with FullCalendar.js.
- **Task Manager**: Add/remove to-do items with persistence.
- **AI Assistant**: Ask questions using Google's Gemini API.
- **Responsive Design**: Works on desktop and mobile.
